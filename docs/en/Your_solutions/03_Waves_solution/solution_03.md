# Solution 03 — Superposition

## 1. Given waves

y₁ = A sin(kx − ωt)  
y₂ = A sin(kx + ωt)

---

## 2. Superposition

Use identity:

sin a + sin b = 2 sin((a+b)/2) cos((a−b)/2)

---

## 3. Result

y = 2A sin(kx) cos(ωt)

---

## 4. Nodes

Nodes occur when:

sin(kx) = 0

kx = nπ

x = nλ/2

---

## Final answer

Standing wave:

y(x,t) = 2A sin(kx) cos(ωt)
