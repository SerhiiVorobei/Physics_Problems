# Solution 08 — Traveling Wave Check

## a)
y(x,t) = A cos(kx² − ωt)

Not a traveling wave because the argument is nonlinear in x (x²).

---

## b)
y(x,t) = A(x − vt)²

Yes, this is a traveling wave of the form f(x − vt).

---

## c)
y(x,t) = A log(x + vt)

Yes, this is a traveling wave of the form f(x + vt).

---

## Final answer
a) No  
b) Yes  
c) Yes
