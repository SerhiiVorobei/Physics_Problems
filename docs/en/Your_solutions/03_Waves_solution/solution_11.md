# Solution 11 — Two-Slit Interference

## Total wave

u = u1 + u2

---

## Sources

u1 = A/r1 · sin(kr1 − ωt)  
u2 = A/r2 · sin(kr2 − ωt)

---

## Key condition

Δr = r2 − r1

---

## Interference

Constructive:
Δr = nλ

Destructive:
Δr = (n + 1/2)λ

---

## Parameters

User controls:
- slit distance d
- wavelength λ

---

## Result
Interference pattern forms alternating bright and dark fringes.
