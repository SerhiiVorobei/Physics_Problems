# Solution 11 — Time-dependent force

## 1. Given

m = 3 kg

F = (15t, 3t − 12, −6t²)

---

## 2. Acceleration

a = F/m

a = (5t, t − 4, −2t²)

---

## 3. Velocity

Integrate:

v(t) = ( (5/2)t² + 2, (1/2)t² − 4t, −(2/3)t³ + 1 )

(using initial v₀)

---

## 4. Position

Integrate again:

r(t) = (
(5/6)t³ + 2t + 5,
(1/6)t³ − 2t² + 2,
−(1/6)t⁴ + t − 3
)

---

## 5. Interpretation

Motion is fully deterministic under time-dependent force.
