# Solution 03 — Biot-Savart Law

## 1. Formula

B = (μ₀ I / 4π) * (L sinθ / r²)

θ = 90°

---

## 2. Given

I = 3 A  
L = 0.1 m  
r = 0.2 m  

---

## 3. Calculation

B = (10⁻⁷ × 3 × 0.1) / (0.2²)

B = (3×10⁻⁸) / 0.04

B = 7.5×10⁻⁷ T

---

## Final answer

B ≈ 7.5×10⁻⁷ T
