# Solution 07 — Wavelength & Frequency

## 1. Given

λ = 550 nm = 5.5×10⁻⁷ m

---

## 2. Frequency

f = c / λ

f = 3×10⁸ / 5.5×10⁻⁷

f ≈ 5.45×10¹⁴ Hz

---

## 3. Color

550 nm → green light (yellow-green region)

---

## Final answer

- Color: green  
- Frequency: ≈ 5.5×10¹⁴ Hz
