# Solution 14 — Hadamard gate

## Operation

H|0⟩ = (1/√2)(|0⟩ + |1⟩)

---

## Result

|ψ⟩ = 1/√2 |0⟩ + 1/√2 |1⟩

---

## Explanation

This is a superposition state.

It represents simultaneous probability amplitudes of both states.

---

## Schrödinger cat interpretation

State is not classical — system is in linear combination of two outcomes until measurement.
