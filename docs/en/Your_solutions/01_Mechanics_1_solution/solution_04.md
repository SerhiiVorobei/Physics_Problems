# Solution 04 — Vector Calculus

## 1. Given position vector

r(t) = (3t²)i + (5t − 8t²)j

---

## 2. Velocity vector

Velocity is the first derivative of position:

v(t) = dr/dt

Differentiate each component:

x(t) = 3t²  →  vₓ(t) = 6t  
y(t) = 5t − 8t²  →  vᵧ(t) = 5 − 16t

So:

v(t) = (6t, 5 − 16t)

---

## 3. Acceleration vector

Acceleration is derivative of velocity:

a(t) = dv/dt

vₓ(t) = 6t  →  aₓ(t) = 6  
vᵧ(t) = 5 − 16t  →  aᵧ(t) = −16

So:

a(t) = (6, −16)

---

## 4. Interpretation

- Acceleration is constant in time
- Motion is uniformly accelerated in both directions
- Horizontal acceleration is positive, vertical is negative
